## üöÄ Quick Start Guide

### 1. Clone Repository
```bash
git clone https://github.com/crpro2025/trial-transport.git
cd trial-transport
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Setup Environment
```bash
cp .env.example .env
# Edit .env with your credentials
```

### 4. Setup Database
```bash
# Generate Prisma client
npx prisma generate

# Run migrations
npx prisma migrate dev --name init

# Seed demo data
npm run prisma:seed
```

### 5. Start Development
```bash
npm run dev
```
